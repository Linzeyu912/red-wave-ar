# S1 紧凑地下储洞小室：用户指定修订（M3D-01R）

> 状态：`USER_APPROVED_REVISION / WHITEBOX_AND_STYLE_REVIEW_REOPENED`
> 决定日期：2026-07-20
> 优先级：高。本文件在 S1 空间尺度、布局和电台审核要求上覆盖此前的 8 m × 8 m 白盒口径；不覆盖项目的“纯虚拟展馆、非 1:1 复原、无铭文通用复原”边界。

## 1. 用户决定

1. 已交付的 8 m × 8 m 白盒显得过大、过空，停止沿用；
2. S1 改为“地下储洞门内侧的一间小型工作室”，而不是多分区的大型展馆；
3. 电台由粗略箱体升级为**审核级通用电台复原**，让手机屏幕上能读出“无线电设备正在工作”的结构；
4. `USER-VIS-003` 只作为内部观察参考，用于提取电台、发报键、线缆、木桌、煤油灯和紧凑地下工作氛围；不得复制图片、文字、品牌、展陈布局或任何具体设备型号。

“地下储洞”是原创虚拟场景叙事，不表示真实战时空间的测绘、原址或 1:1 复原。

## 2. 新的空间规格（白盒与风格审核唯一基准）

| 项目 | 要求 |
|---|---|
| 室内净尺寸 | 约 **5.3 m（X）× 4.7 m（Z）× 2.25 m（净高）**；允许 ±0.2 m 的建模调整 |
| 房间性质 | 低矮、紧凑、可工作的地下储洞小室；不做地下基地、长隧道或展览大厅 |
| 墙体 | 厚度约 0.20–0.30 m；以夯土、粗石、局部灰砖修补与旧木门框表达，不使用湿亮钟乳石洞壁 |
| 入口 | 南侧偏西的低矮储洞门/旧木门片，洞口清晰可见但不需要可穿行的长通道；不复制现馆门楼 |
| 行走区 | 建议 `x [-2.25, 2.25]`、`z [-1.95, 1.95]`；桌两侧和后方的可行走净宽均不得低于 1.2 m |
| 信息层 | 不再占据整面大墙；只保留一块小型无文字导览牌或由 Compose 信息卡承载时间线/路线说明 |
| 氛围 | 桌面局部暖光，洞壁与门口低亮但可辨；克制、干燥、安静、可参观，不能阴森或恐怖 |

## 3. 布局与交互基准

```text
                北 / 低矮夯土或石墙
┌──────────────────────────────────┐
│      小型导览牌（非整墙展板）    │
│                                  │
│        木桌：radio / key / 资料包 │
│        暖光操作核心              │
│                                  │
│  旧木门/储洞入口       P0 游客起点│
└──────────────────────────────────┘
                南 / 入口侧
```

- `visitor_start`：建议 `[0.0, 1.55, 1.65]`，面向负 Z 的木桌；最终 `rotation_deg` 以真机轴向验证为准；
- 木桌：中心建议 `[0.0, 0.0, 0.10]`，尺寸保持 `1.80 × 0.75 × 0.78 m`；
- `p_s1_radio`：桌面右后，建议世界坐标 `[0.32, 0.78, 0.04]`；
- `p_s1_key`：桌面前中，建议 `[-0.34, 0.78, 0.34]`；
- `p_s1_codebook`：桌面左后，建议 `[-0.58, 0.78, -0.10]`；
- 第一眼到 radio 的水平距离建议 `1.4–1.9 m`，三个 MVP 文物均应在 `1.0–2.5 m` 点击带内；
- 可以放一只矮凳、一盏原创煤油灯和一只无铭文木箱；不得用大量箱子、墙面信息或装饰把房间再撑成“展厅”。

## 4. `p_s1_radio` 审核级优化要求

### 必须有

- 无品牌、无文字、无序列号的深军绿/暗褐绿色便携箱式外壳；建议约 `0.42 × 0.28 × 0.32 m`；
- 前部略倾斜的控制面，至少有一个无可读数值的表窗、5–7 个大小有差异的旋钮、插孔与接线端；
- 1 根深色布包线或橡胶线，从接线端自然下垂/绕向桌面，不能穿模或悬浮；
- 机身、面板、旋钮、表窗和接线端必须在 720p 预览中有清楚的层次，不做成单一方块；
- 节点至少保留：`radio_body`、`dial_main`、`meter_window`、`knob_01...`、`cable_socket_01...`；
- 中等复杂度即可：主模型建议 2.5k–5k 三角面，优先轮廓、面板层级与移动端清晰度，不做不可见内部电路。

### 必须避免

- 真实历史型号、品牌、徽章、军队番号、频率刻度、英文/中文铭文、可读仪表数据；
- 对 `USER-VIS-003` 或纪念馆展品逐轮廓临摹；
- 发光按键、科幻蓝光、过度锈蚀、血迹、破坏感和“道具化玩具外观”；
- 以高模细节替代可读轮廓；所有细节应服务于 0.8–2.0 m 手机观看距离。
