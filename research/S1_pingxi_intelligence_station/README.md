## 1. 资料包用途

本目录把平西情报联络站相关史料转换为可被内容模型、建模模型和编码模型共同使用的约束。它不是一份可以直接照抄的“百科介绍”，而是一个带来源、证据等级、版权状态和待确认项的制作输入包。

本轮只完成 S1。用户审核通过前，不把同样的方法批量复制到其他地点，也不把白盒占位资产升级为正式历史复原资产。

## 2. 已确认结论

1. **S1 应定位为“有史料依据的虚拟主题展馆”，不宣称现址或战时空间的 1:1 复原。** 现有来源能够确认组织、任务、路线、部分人物和涧沟村秘密电台故事，但不能确认战时房间的完整尺寸、陈设和设备型号。
2. **1941 年与 1943 年必须分开表述。** 1941 年 2 月是平西情报交通联络站体系成立时间；权威资料同时记载站址初设于涞水计鹿村。涧沟村秘密电台有多条来源明确到 1943 年冬。不能把两个时间点合并成“1941 年涧沟村电台室建成”。
3. **物理叙事时间点确认为 1943 年冬。** 以涧沟村山洞秘密电台和农家掩护关系作为空间核心；1947 年清风店战役情报传递作为结果层内容，通过展板、音频或时间线呈现，不把 1943 与 1947 的物件混成同一时刻的写实场景。
4. **电台和发报键进入正式方向研究；技术 ID `p_s1_codebook` 保留，但不再把它表现为有具体密码内容的历史密码本。** 用户可见名称确认为“电文与保密通信资料包”，造型采用无铭文纸束/资料夹的通用复原。
5. **现有外观照片和采访视频只能作为观察参考。** 未发现开放授权。资料包只登记链接和观察结论，不把图片、视频帧或地图复制进仓库，不允许直接用作纹理、展板或宣传图。

## 5. 审核状态约定

| 状态 | 含义 | 是否允许进入正式资产 |
|---|---|---|
| `VERIFIED` | 有足够权威来源，可写入正式内容 | 是 |
| `SUPPORTED_WITH_CAVEAT` | 基本可信，但必须保留限定语或时间边界 | 是，需带限定语 |
| `VISUAL_REFERENCE_ONLY` | 只证明当前展陈或外观，不证明战时原状 | 只能做风格参考 |
| `UNVERIFIED` | 尚无可靠证据或来源存在冲突 | 否 |
| `OWNER_APPROVAL_REQUIRED` | 涉及叙事选择、权利或产品取舍 | 否，先由用户决定 |

## 6. 用户补充素材的接收规则

用户后续提供照片、视频、扫描件、讲解词或馆方资料时，不直接覆盖本资料包的结论。每份素材先登记：

- 文件名或链接；
- 提供者与原始来源；
- 拍摄/出版时间；
- 素材显示的对象和地点；
- 是否允许研究、内部建模参考、应用内展示和宣传使用；
- 是否存在人物肖像、馆内拍摄限制、第三方图片或未公开信息；
- 可支持或推翻的事实 ID。

完成登记和复核后，再更新 `verified_facts.md`、视觉台账和建模冻结状态。
