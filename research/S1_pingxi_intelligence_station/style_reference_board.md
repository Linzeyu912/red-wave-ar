# S1 风格参考板与场景约束

> 状态：`VISUAL_DIRECTION_FROZEN_FOR_NEXT_ASSET_PASS`

> 适用阶段：白盒审核、`M3D-02` 核心文物、`M3D-03` 正式环境
> 重要边界：本文件冻结的是**原创虚拟主题展馆的风格方向**，不是战时原貌结论，也不构成图片使用授权。

## 1. 一句话方向

**用“普通山村院落的掩护外壳、紧凑山洞电台的行动核心、克制清晰的现代信息层”构成 S1；整体隐蔽、朴素、低饱和，但不能黑、脏、恐怖或像军事基地。**

建模时始终保留以下三层区别：

1. **地域层**：妙峰山山谷聚落、石阶、灰砖、石基、旧木，只负责说明“为什么这里便于隐蔽和联络”；
2. **行动层**：1943 年冬涧沟村秘密电台的桌面操作组，是第一视觉中心；
3. **展馆层**：时间线、路线示意、热点和信息卡属于当代解释系统，材质和亮度必须明显区别于历史演绎物。

## 3. 视觉关键词与反关键词

### 必须呈现

- 普通、隐蔽、紧凑、克制；
- 山地农家材料，而非纪念性建筑材料；
- 暖光集中在“正在工作”的桌面，外围保持低亮但可辨；
- 电台、发报键、资料包之间有清楚的操作关系；
- 现代信息层轻、薄、安静，能够被识别为解释系统；
- 360° 环视时四个方向都有层级，但首屏只有一个主角。

### 禁止呈现

- 宏大的革命历史展馆、纪念堂、宫殿或牌楼；
- 规则混凝土地堡、军事指挥中心、成排机柜；
- 科幻控制台、霓虹赛博空间、全息蓝色“科技线”；
- 恐怖山洞、牢房、刑讯室、血迹、强烟雾或频闪警报；
- 满场红旗、党徽、标语、口号墙和高饱和“中国红”；
- 当前纪念馆的门楼、展柜、雕塑、浮雕和展墙复刻；
- 把 1943、1945、1947 的物件做成同一历史瞬间；
- 把“地下情报”误解为大型地下基地。

## 4. 色彩系统

下列色值是原创场景的**近似工作色**，不是从受版权图片逐像素取色。允许在材质和曝光测试后小幅调整，但不得改变冷暖层级。

| 角色 | 建议色 | 使用范围 | 上限/限制 |
|---|---|---|---|
| 山石深灰 | `#3F4542` | 洞壁、暗部结构、远端体量 | 不能压成纯黑 |
| 灰砖 | `#626965` | 地域入口、局部墙片 | 不做均匀新砖墙 |
| 粗石 | `#898A80` | 石基、地面边缘、台阶片段 | 避免大面积高亮白石 |
| 旧木深褐 | `#49392E` | 梁、门框、桌凳 | 木纹尺度必须可信 |
| 土褐 | `#6F5842` | 夯土、纸包、局部地面 | 不做橙黄“古风滤镜” |
| 低饱和军绿 | `#4D5643` | 通用箱式电台、少量设备 | 不宣称历史设备原色 |
| 暗朱红 | `#78352F` | 入口木构、极少量导览强调 | 基础场景可见面积建议不超过 8–12% |
| 暖灯琥珀 | `#C58B4A` | 桌面任务光、油灯附近 | 只形成局部焦点 |
| 信息层灰蓝 | `#708489` | 路线/时间信息面、热点中性态 | 禁止大面积霓虹蓝 |
| 纸张暖灰 | `#B8AA91` | 无铭文资料包 | 不烘焙可读档案内容 |

色彩验收：缩小到手机屏幕宽度后，应先看到暖色电台桌，再读到灰蓝信息层，最后感知灰砖/山石环境；红色不能成为第一视觉中心。

## 5. 材质与表面规则

### 5.1 环境材质

- **山石/夯土**：中高粗糙度、低镜面、法线起伏尺度偏大；不使用湿亮洞壁和密集尖刺。
- **灰砖**：允许尺寸、色差和边缘磨损轻微变化；禁止每块砖完全随机、破损过量或像废墟。
- **石基/石阶**：不规则但稳定，边缘磨圆；只做地域意象，不做可攀爬主路线。
- **旧木**：深褐哑光，保留使用痕迹；禁止鲜红清漆、金色彩绘和复杂雕花。
- **地面**：压实土/粗石过渡为主，操作区必须平整，不能让桌椅产生视觉倾斜。

建议 PBR 粗糙度范围：山石 `0.80–0.95`，灰砖 `0.75–0.90`，旧木 `0.65–0.85`，设备喷漆金属 `0.45–0.65`。这是工作范围，不是历史测量值。

### 5.2 文物材质

- 电台采用哑光深绿或深灰箱体、深色旋钮和低反射刻度窗；不做崭新军品，也不做锈蚀报废品。
- 发报键以深色金属、绝缘旋钮和简洁底座构成；尺寸可读性优先于微小机械细节。
- “电文与保密通信资料包”采用暖灰纸束、暗布带或普通资料夹；不得出现标题、印章、密码表和手写内容。
- 金属边缘磨损应克制，只在高频接触处出现；不得使用统一“脏污生成器”覆盖全部资产。

### 5.3 信息层材质

- 时间线和路线面采用薄片、局部透明或低强度自发光，保持平整、精确、现代；
- 历史演绎物使用不自发光的实体材质，二者不能混淆；
- 大段文字由 App UI 展示，3D 中只保留标题级短标签和图形节点；
- 交互轮廓由运行时控制，不能永久烘焙在资产上。

## 6. 空间构图

沿用 `scene_layout_brief.md` 的坐标、8 m × 8 m 白盒、游客起点和通道，不因为风格图改变已验证的交互布局。

### ZONE-A 地域入口

- 只做“灰砖墙片 + 粗石基 + 暗红木框 + 远山/屋面抽象剪影”的压缩组合；
- 不建立完整现馆正门，不挂真实馆名或楹联；
- 高差通过墙脚、地面纹理和远景剪影表达，不在主通道放陡阶；
- 入口只占辅助视线，用户出生后不需要回头才能理解地点。

### ZONE-B 秘密电台操作区

- 保持桌宽 1.8 m、深 0.75 m、高 0.78 m 的已定占位基准；
- 电台置后、发报键置前、资料包置侧，三者轮廓互不遮挡；
- 桌面是唯一强暖光区，背景比主体低 1–2 个视觉层级；
- 墙面/洞壁在桌后形成稳定暗框，不设置大型窗、旗帜或口号；
- 不放写实人物。若以后确需表现操作姿态，优先用短动画、影子或音频，不复制现馆雕塑。

### ZONE-C 路线与时间线

- 用灰蓝线、节点和清晰时间刻度表达 1941、1943、1945、1947；
- 1943 节点可与桌面暖光呼应，1947 只在结果层出现；
- 不使用历史地图截图、当代地图底图或现馆展板照片；
- 线条不能像雷达、作战指挥 UI 或科幻 HUD。

### ZONE-D 结果与纪念区

- 视觉安静，亮度低于路线区；
- 可用少量不带姓名的光点/纸页轮廓表达“有名与无名的隐蔽战线人员”；
- 不做英雄雕像、姓名墙复制或煽情舞台追光。

## 7. 灯光与氛围

- 环境基调：1943 年冬的冷静、干燥、低饱和感；现有春夏照片不能决定场景季节。
- 主任务光：桌面局部暖光，建议 `2400–3000 K`；软边、无闪烁、不可过曝。
- 环境补光：中性偏冷，建议 `3800–4600 K`；保证墙角、地面边界和出口可辨。
- 信息层：灰蓝低亮度，不与桌面争夺焦点；自发光只达到可读，不制造光污染。
- 不使用体积雾、粒子灰尘、强暗角、镜头光斑和高频动态阴影作为主要氛围手段。
- 从游客起点截图转为灰度后，桌面仍应是最大明度聚焦区；最暗可行走边界仍可分辨。

## 8. 移动端与类 VR 约束

- 地域高差只作视觉提示，主通道净宽继续保持 `≥ 1.2 m`；
- 地面法线和视差不能强到造成“地面起伏”错觉；
- 避免贴脸悬挂物、低梁和快速摆动物；
- 任何装饰不得遮挡 `p_s1_radio`、`p_s1_key`、`p_s1_codebook` 的射线拾取体；
- 360° 环视时不需要持续仰头，主要信息中心保持在 `1.2–1.8 m`；
- 场景第一帧和回正后的中央 40% 区域必须能看到电台操作组；
- 若材质细节与帧率冲突，优先保留轮廓、明暗层级和操作关系，删除小型装饰。

## 9. 建模模型执行规则

1. 白盒已通过前，不因本风格板重做几何布局；只在预览中验证材质分区和明暗方向。
2. 进入中模前先提交 3 张灰模/简单材质图：游客起点、操作区 45°、路线区回望。
3. 三张图必须附“采用的参考编号 + 没有复制的说明 + 尚未确认的造型列表”。
4. 用户审核风格图后，才制作电台/发报键细节和正式环境材质。
5. 所有第三方材质、HDRI、贴图和字体进入 `asset_report.csv`；没有清晰许可的一律不用。
6. 禁止把本文件的外链图片保存进 Blender 工程、贴图目录、参考图打包或 GLB extras。
7. 不自行补电台型号、品牌、铭牌、频率、地图地名、历史人物肖像和可读档案。
8. 发现参考图与史实文件冲突时，以 `verified_facts.md` 和 `source_register.md` 为准，并停止冲突部分。

## 11. 风格验收清单

- [ ] 整体能用“普通山村掩护 + 紧凑电台行动 + 克制信息层”概括。
- [ ] 没有把当前纪念馆、关帝庙、农家院和山洞合并成“原址复原”。
- [ ] 没有复制真实馆名、楹联、旗阵、护栏、展板、雕塑或玻璃展柜。
- [ ] 红色面积不超过基础场景约 8–12%，且不是第一视觉中心。
- [ ] 游客起点第一眼看见电台操作组。
- [ ] radio、key、资料包轮廓互不遮挡。
- [ ] 1947 内容只在结果/信息层，不混入 1943 写实陈设。
- [ ] 没有精确型号、品牌、军队编号、频率、密码表、印章或真实电文。
- [ ] 暗部可辨，没有纯黑通道、频闪、体积雾或恐怖化处理。
- [ ] 灰砖、山石、旧木的粗糙度和纹理尺度相互区分。
- [ ] 历史演绎物不自发光，现代信息层低强度自发光，交互高亮由程序控制。
- [ ] 主通道净宽、游客起点、交互距离和碰撞仍符合 `scene_layout_brief.md`。
- [ ] 所有第三方资产有许可记录，外链参考图未进入工程和运行包。
- [ ] STYLE-01、STYLE-02、STYLE-03 三张审核图齐全并通过用户确认。

## 13. 用户补充图片的影响

`USER-VIS-001/002` 没有改变冻结方向，只让以下规则更具体：

- 灰砖不能做成单一均匀色，应有克制的明度、灰度和修缮痕迹变化；

- 三张 STYLE 审核图不得出现真实牌匾、楹联、旗帜、栏杆、浮雕题字或浮雕人物；
- 建模模型读取原图时必须遵守 `modeling_input/S1/03_MATERIALS_GUIDE.md`，原图不得进入工程或远程仓库。
